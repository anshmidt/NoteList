# NoteList
NoteList is an Android app for managing notes and organizing them in lists.
![notelist screenshots](https://user-images.githubusercontent.com/12444628/222925488-abc44943-4e93-4c6f-9f42-9faa32e7f93f.png)


## Used technologies:
- Jetpack Compose 
- MVVM 
- Kotlin Coroutines
- Kotlin Flow 
- Room 
- Jetpack DataStore 
- Koin
